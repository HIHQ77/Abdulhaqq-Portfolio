Educational Background

I'm currently pursuing a Bachelor of Science in Computer Science, with coursework spanning web technologies, data structures, and discrete mathematics. COS 106 introduced me to the fundamentals of building for the web — HTML structure, CSS layout, and JavaScript interactivity — which this very site puts into practice.

Outside of core requirements, I gravitate toward electives in design and human-computer interaction, where I get to think about not just whether software works, but whether it feels good to use.
